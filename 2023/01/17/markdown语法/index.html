2. 文字效果

markdown支持使用* _等特殊符号来进行文字效果。

也支持使用html标签来设置文字效果

  1. 斜体

    1. *1* 效果： 1
    2. _1_ 效果： 1
    3. <i>1</i> 效果：1
    4. 总结：
      1. *或_的效果是一致的，两个*或_将需要斜体的内容包裹起来
      2. 直接使用html i标签，用法与*的用法一致，使用i标签将内容包裹起来

  2. 粗体

    1. **1** 效果：1
    2. __1__效果：1
    3. <b>1</b>效果：1
    4. 总结：
      1. 与斜体的用法一致，只不过不是使用两个*包裹内容了，而是改为了四个，即内容两侧各有两个
      2. 标签语法就与斜体的一致

  3. 粗斜体

    1. ***1*** 效果：1
    2. ___1___效果：1
    3. 总结：
      1. 与斜体/粗体的用法一致，内容两侧各有三个*

  4. 删除线

    1. <del>delete</del> 效果： delete
    2. ~~delete~~ 效果：delete
    3. 总结：
      1. 使用标签del
      2. 使用~~双波浪线

  5. 下划线
